# engagement survey

## DEMO

## 紹介と使い方

  - チームメンバーの満足度調査を行うツール
  - 質問ページ（従業員用）と結果ページ（管理者用）の構成
  - チームメンバーに質問ページのリンクを送り満足度調査の回答をしてもらう→結果ページで集計結果を確認し、今のチームの状態を可視化することができる。

## 工夫した点

  - 卒制を視野に入れたプロダクト。前回課題で作成したものをfirebaseからデータベースに移行した。
